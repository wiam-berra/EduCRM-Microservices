# 🎓 EduCRM – Intelligent Academic Management Platform

An intelligent Academic Management Platform developed as a Final Year Project using a modern Microservices Architecture. The platform provides efficient management of students, courses, attendance, academic performance, and AI-powered academic risk prediction.

---

This project was collaboratively developed by:

## 📌 Project Overview

EduCRM is a web-based academic management platform designed to simplify university administration while integrating Artificial Intelligence to improve student monitoring and decision-making.

The application follows a Microservices Architecture to ensure scalability, maintainability, and flexibility.

## 🏗️ System Architecture

The project follows a Microservices Architecture composed of multiple independent services.

- API Gateway
- Eureka Discovery Server
- Authentication Service
- Student Service
- Course Service
- Attendance Service
- AI Service
- Chatbot Service

Each service communicates using REST APIs.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/wiam-berra/EduCRM-Microservices.git
```

### Backend

```bash
cd auth-service
mvn spring-boot:run
```

Repeat for the remaining services.

### Frontend

```bash
cd frontend
npm install
npm run dev
```
